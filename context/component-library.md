## 典型映射
- text -> 共享输入组件
- enum -> 共享下拉组件
- boolean -> 共享开关组件
- date -> 共享日期组件
- table -> 共享表格组件
- textarea -> 共享多行输入组件
- upload -> 共享上传组件

## 维护建议
- 映射尽量写成“字段类型 -> 推荐组件”
- 如果有多个可选组件，明确默认优先级
- 如果某类字段还没有统一组件，明确写“当前未统一”

## 禁止事项
- 不要在这里记录一次性页面实现细节
- 不要把不存在的组件写成既有能力
